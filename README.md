# AirCheck
😷  Coroutine, Retrofit2, OpenAPI를 이용한 우리 동네 미세먼지 측정앱 😷

## 개발환경
* OS : windows 10 pro
* IDE : Android Studio
* Language : Kotlin

## 사용한 것
* LocationManager
* Retrofit2
* Coroutine
* AppWidgets
* lifecycle
* OpenApi (카카오, 공공데이터)
